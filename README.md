# Saucedemo Test Unit<br>

**Description:**<br>
A test unit for testing logging functionality for Saucedemo.com.<br>

**Key features:**<br>
 - A test unit that tests the front end of the login functionality for Saucedemo.com<br>
 - The test unit also tests for basic security issues that usually arise with improper validation or sanitation of user input.<br>
 - Test unit also checks that valid and invalid user input leads to predictable output and provides secure and meaningful error messages.<br>

 - git clone [repository](https://github.com/KateYurch/Cypress_Test_Unit_Saucedemo.git)
 - npm cypress open<br>
